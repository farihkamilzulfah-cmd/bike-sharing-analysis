# Proyek Analisis Data: Bike Sharing Dataset 🚲

## Deskripsi
Proyek ini merupakan analisis terhadap dataset penyewaan sepeda. Fokus utama analisis ini adalah memahami bagaimana faktor musiman dan perbedaan antara hari kerja serta hari libur memengaruhi jumlah penyewaan sepeda. Dashboard interaktif dibangun menggunakan Streamlit untuk mempermudah eksplorasi data.

## Struktur Folder
- **/Dashboard**: Berisi file utama dashboard (`dashboard.py`) https://bike-sharing-analysis-eijcyknqqfwgevk5khpz7v.streamlit.app/ dan dataset yang telah dibersihkan (`main_data.csv`).
- **/Data**: Berisi dataset asli sebelum diproses (`day.csv` dan `hour.csv`).
- `Farih_Kamil_Zulfah_Proyek_Analisis_Data.ipynb`: Notebook yang berisi seluruh proses analisis data (Gathering, Assessing, Cleaning, Explanatory Analysis).
- `requirements.txt`: Daftar pustaka (library) yang diperlukan untuk menjalankan proyek.
- `README.md`: Dokumentasi proyek.
